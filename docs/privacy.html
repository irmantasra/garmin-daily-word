Daily Word — Privacy Policy

Last updated: 2026-07-10

Daily Word is a Garmin Connect IQ app that displays the day's Catholic Mass reading references (in Lithuanian or English). This policy explains what data the app handles.

What data we collect

None. The app does not collect, store, or transmit any personal information. It does not access your location, health metrics, activity data, contacts, or device identifiers.

Network use

The app makes one type of network request: it downloads a small public JSON file containing that day's reading references (for example Mt 10, 1-7) from a static website hosted on GitHub Pages. This request sends no personal data — only a standard HTTP GET for a public file. No account, login, or tracking is involved.

Data storage

The most recently downloaded reading references are cached locally on your watch so the app can display them quickly. This cache contains only public scripture references and never leaves your device.

Third parties

Reading references are sourced from publicly available liturgical calendars (lk.katalikai.lt for Lithuanian, bible.usccb.org for English). The app itself shares no data with any third party.

Contact

Questions about this policy can be raised via the project's issue tracker on GitHub.

Daily Word is an independent app and is not affiliated with Garmin, the USCCB, or Katalikai.lt.
